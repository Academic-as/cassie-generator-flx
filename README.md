**C.A.S.S.I.E. Generator** — это интуитивно понятный и визуально отточенный инструмент для администраторов **FLX Project**. Он превращает рутинное написание CASSIE в мгновенную команду!

> [!TIP]
> [Опробовать генератор в браузере](https://Academic-as.github.io/cassie-generator-flx/)

---

## ✨ Возможности и функционал

| Функция | Описание |
| :--- | :--- |
| **🇺🇸 ⇄ 🇷🇺 Трансляция** | Автоматический перевод русского текста в синтаксис CASSIE. |
| **🎨 Дизайн** | Работа с цветом через HEX и пресеты прямо в интерфейсе. |
| **🎵 Звук** | Регулировка `$pitch` и `$sleep` ползунками (без ручного ввода цифр). |
| **🛠 Шаблоны** | Библиотека команд для MTF, NATO и праздничных ивентов. |
| **🔍 Поиск** | Интеллектуальная система подсказок при отсутствии слова в базе. |

> [!WARNING]  
> Система «Трансляция» находится в процессе постоянного улучшения. В «russianToCassie» некоторые русские слова могут и будут содержать ошибки.

---

## 🚀 Механика работы

Инструмент автоматизирует создание строк формата: `cassie sub [EN] csub [RU]`

1. **Ввод:** `Внимание всему персоналу зоны 02`
2. **Обработка:** Система преобразует числа и переводит ключевые слова.
3. **Результат:** 

```bash
cassie sub Attention All Personnel Zone 0 2 csub Внимание всему персоналу зоны 02
```

## 📖 Команды форматирования

Используйте встроенные функции для создания уникальной атмосферы оповещений:

| Команда | Описание | Эффект |
| :--- | :--- | :--- |
| `<split>` | **Разделитель** | Создает логическую паузу в тексте. |
| `$STUTTER` | **Эффект заикания** | Не до конца понятно, как это работает |
| `.g1` — `.g7` | **Сигналы** | Дополнительные звуки: гудки, визги и алерты 079. |
